Guia Rápido

Bem-vindo à Calculadora de Rank! Este guia irá ajudá-lo a usar esta ferramenta de maneira simples e rápida. Siga os passos abaixo:

Entrada de Dados: No topo, você encontrará campos para inserir valores em diferentes categorias, como "Alma", "Mana", "Força" e mais. Insira os valores que deseja calcular.

Rank e Média: À medida que insere os valores, o "Rank" será atualizado automaticamente com base nas médias. O "Rank" indica sua classificação com base nos valores inseridos.

Total de Pontos: Abaixo dos campos de entrada, você verá o "Total de Pontos". Ele mostra a soma dos valores inseridos em todas as categorias.

Valores Específicos: Para cada categoria, como "Alma" ou "Mana", você verá os valores totais calculados com base nos pontos de experiência (XP) inseridos. Isso ajuda a entender como suas escolhas afetam os atributos específicos.

Dicas Adicionais: Use esta ferramenta para planejar estrategicamente como distribuir seus pontos de experiência para obter os melhores resultados.

Lembre-se, insira apenas números válidos nas caixas de entrada e observe como suas escolhas afetam médias, ranking e atributos específicos. Experimente combinações diferentes para alcançar o resultado desejado. Divirta-se calculando seu Rank!
